#🎮 Poketmon Console Battle

C++와 Windows Console API를 활용해 포켓몬 스타일의 턴제 배틀 게임을 개발했습니다. 
포켓몬·기술을 클래스로 모델링하고, 속성 상성, 물리/특수 데미지, 명중 및 급소 판정, 6종의 상태 이상, 능력치 랭크, 파티 교체, 아이템, 경험치와 진화 시스템을 구현했습니다. 
또한 콘솔 좌표와 색상을 직접 제어해 HP/EXP 바와 전투 애니메이션을 표현했습니다.

## 주요 기능

- **턴제 전투**: 기술 사용, 아이템 사용, 포켓몬 교체, 도망치기 선택지를 제공하며 스피드 능력치에 따라 행동 순서를 결정합니다.
- **기술 시스템**: 각 포켓몬은 4개의 기술을 사용합니다. 기술별로 위력, 명중률, 물리/특수 분류, 속성, 상태 이상 확률, 능력치 변화, 회복 효과를 데이터로 관리합니다.
- **데미지 계산**: 공격·방어 또는 특수공격·특수방어 능력치를 구분하고, 자속 보정, 속성 상성, 급소 배율을 조합해 최종 데미지를 계산합니다.
- **상태 이상**: 독, 마비, 화상, 얼음, 수면, 혼란을 구현했습니다. 턴 경과 피해와 행동 불가 확률, 해제 확률 등 상태별 규칙이 전투에 반영됩니다.
- **능력치 랭크**: 공격, 방어, 특수공격, 특수방어, 스피드, 회피 랭크의 상승·하락을 처리하고 전투 능력치에 적용합니다.
- **파티 및 아이템**: 전투 중 포켓몬을 교체하거나 회복약, 부활, 상태 이상 치료제, 스피드 강화 아이템을 사용할 수 있습니다.
- **성장과 진행**: 전투 참여 포켓몬에게 경험치를 분배하고, 레벨 업과 피카츄의 진화 이벤트를 처리합니다.
- **콘솔 UI**: `SetConsoleCursorPosition`과 `SetConsoleTextAttribute`를 이용해 HP/EXP 바, 상태 정보, 메뉴, 포켓몬 ASCII 아트와 스테이지별 타이틀 연출을 출력합니다.

## 조작 방법

- 숫자 키 `1`~`4`: 메뉴 또는 기술 선택
- `Enter`: 선택 확정
- `Backspace`: 이전 메뉴

## 핵심 구현

### 1. 데이터 중심의 포켓몬·기술 설계

`Poketmon` 클래스는 기본 능력치, 현재 HP, 랭크, 상태 이상, 레벨, 경험치와 보유 기술을 관리합니다. `Skill` 클래스는 공격에 필요한 값뿐 아니라 부가 효과까지 하나의 데이터 모델로 표현합니다. 새로운 포켓몬과 기술을 목록에 추가해 기존 전투 로직에서 재사용할 수 있도록 구성했습니다.

### 2. 단계별 데미지 계산

```text
공격/특수공격 × 기술 위력 ÷ 방어/특수방어
        × 속성 상성 × 자속 보정 × 급소 보정
```

물리·특수 기술은 서로 다른 능력치를 참조하며, 현재 HP 비례 기술은 별도의 계산 경로로 처리합니다. 이후 HP를 갱신하고 상성 및 급소 결과를 전투 메시지로 출력합니다.

### 3. 확률과 상태를 포함한 턴 처리

한 턴 안에서 명중 판정, 상태 이상에 따른 행동 가능 여부, 기술의 부가 효과, 랭크 변화, 회복, 전투 불능 여부를 순서대로 처리합니다. 플레이어와 상대의 스피드를 비교해 선공을 정하고, 교체와 아이템 사용도 하나의 행동으로 반영합니다.

### 4. 좌표 기반 콘솔 렌더링

일반적인 순차 출력 대신 화면 영역별 좌표를 정해 필요한 정보만 다시 그립니다. HP와 경험치는 현재 값에 비례한 막대로 표현하고, 상태 이상과 선택 메뉴는 색상을 달리해 텍스트 환경에서도 전투 상황을 빠르게 파악할 수 있도록 했습니다.
